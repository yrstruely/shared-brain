# SpecFlow

## Basic Information
- Type: product
- Source: [[sources/bdd-cucumber-gherkin-reference_7dfdd4]]

## Description
SpecFlow is a .NET Behaviour-Driven Development ([[concepts/Behaviour-Driven Development|BDD]]) framework inspired by Cucumber. It allows .NET developers to write BDD scenarios using [[concepts/Gherkin|Gherkin]] feature files and implement them with [[concepts/Step Definition|step definitions]] in languages like C#. The source presents SpecFlow as part of a family of Cucumber-based BDD tools, alongside Python's Behave and PHP's Behat. By integrating into the .NET ecosystem, SpecFlow enables teams to apply declarative scenario writing and domain-driven conversations within Microsoft's development environment. It is a key tool for .NET organizations adopting BDD practices.
