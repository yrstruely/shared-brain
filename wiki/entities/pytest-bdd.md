# pytest-bdd

## Basic Information
- Type: product
- Source: [[sources/bdd-cucumber-gherkin-reference_7dfdd4]]

## Description
pytest-bdd is a BDD plugin for the Python testing framework pytest, allowing teams to adopt Behaviour-Driven Development practices without switching to a separate test runner. It enables Python developers to write Gherkin scenarios and connect them to step definitions while leveraging pytest’s fixtures, assertions, and reporting capabilities. The tool is listed alongside other BDD frameworks such as [[entities/behave|Behave]] (Python), [[entities/specflow|SpecFlow]] (.NET), and [[entities/cucumber|Cucumber]] (multi-language), highlighting the diversity of available implementations. pytest-bdd is a pragmatic choice for Python projects that want to incrementally integrate BDD into an existing pytest workflow.
