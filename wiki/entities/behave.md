# Behave

## Basic Information
- Type: product
- Source: [[sources/bdd-cucumber-gherkin-reference_7dfdd4]]

## Description
Behave is a Python BDD (Behaviour-Driven Development) framework that enables development teams to implement BDD practices using Gherkin feature files and step definitions. It is specifically designed for the Python ecosystem, allowing testers and developers to write executable specifications in a declarative, natural-language style. The reference lists Behave alongside other language-specific BDD frameworks such as [[entities/specflow|specflow]] (for .NET) and [[entities/behat|behat]] (for PHP), demonstrating the widespread adoption of BDD principles across different programming environments. With Behave, Pythonistas can follow the same BDD workflow described in the reference—starting with discovery, writing declarative Gherkin scenarios, and automating them with step definitions. It serves as a key tool for applying [[concepts/Behaviour-Driven Development|Behaviour-Driven Development]] in any Python project.
