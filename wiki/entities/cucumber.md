# Cucumber

## Basic Information
- Type: product
- Source: [[sources/bdd-cucumber-gherkin-reference_7dfdd4]]

## Description
Cucumber is the reference Behaviour-Driven Development framework described in the source. It executes automated acceptance tests written in [[concepts/gherkin|Gherkin]], a plain-text language that describes application behaviour in a business-readable format. The tool supports multiple programming languages including Java, Ruby, and JavaScript, and connects Gherkin steps to [[concepts/step-definition|step definitions]] written in code, effectively hard-wiring the specification to the implementation. The source is compiled from Cucumber's official documentation (cucumber.io/docs), establishing it as the authoritative BDD framework in the ecosystem. It is distinct from related tools like SpecFlow or Behave, which are inspired by or based on Cucumber's approach.
