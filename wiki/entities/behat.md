# Behat

## Basic Information
- Type: product
- Source: [[sources/bdd-cucumber-gherkin-reference_7dfdd4]]

## Description
Behat is a Behaviour-Driven Development (BDD) framework for PHP, inspired by [[entities/cucumber|Cucumber]]. It enables PHP developers to write, automate, and execute Gherkin scenarios, bringing the same BDD practices—discovery, formulation, and automation—to the PHP ecosystem. Behat is part of a family of Cucumber-like tools that includes [[entities/specflow|SpecFlow]] (for .NET) and [[entities/behave|Behave]] (for Python), all sharing the [[concepts/gherkin|Gherkin]] language for specifying behaviour. By mapping Gherkin steps to [[concepts/step-definition|step definitions]] in PHP, Behat allows teams to maintain a single source of truth for both specification and test automation, bridging communication between stakeholders and developers.
