# FluentIT Domain Entity

Designs a domain entity through a structured interview with you.

## Usage

```
fluentit-domain-entity --project PROJECT_NAME --feature "feature description"
```

## Example

```
fluentit-domain-entity --project hello-world --feature "user profile management"
```

## Output

- Entity: `backend/src/domain/entities/user.entity.ts`
- Tests: `backend/src/domain/entities/user.entity.spec.ts`
