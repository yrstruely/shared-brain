# FluentIT Frontend Guide

Guides frontend feature implementation from requirements to delivery.

## Usage

```
fluentit-frontend-guide --project PROJECT_NAME --feature FEATURE_NAME
```

## Example

```
fluentit-frontend-guide --project hello-world --feature welcome
```

## Output

- Components, pages, composables, and tests
